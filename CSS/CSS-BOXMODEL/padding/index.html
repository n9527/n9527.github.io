CSS 内边距

元素的内边距是边框和内容之间的距离。单位接收长度单位和百分比，但是不能是负值。

CSS padding 属性定义元素内容和边框之间的空白区域。

  • CSS padding 属性

    如果你希望一个元素的四个内边距都是10px,只需要向下面这样：

    h1 {padding: 10px;}

    还可以按照上、右、下、左的顺序分别设置各个内边距的值，各个边都可以使用不同的长度单位或者百分比：

    h1 {padding: 10px 1em 2em 20%;}

  • 单边内边距属性

    也可以下面四个单独的属性，分别设置上、右、下、左内边距：

    • padding-top
    • padding-right
    • padding-bottom
    • padding-left

  • 内边距的百分比数值

    百分比是参照父级元素的 width 值计算的，上下内边距和左右内边距都是相对父级的 width 而不是 height。
