CSS 浮动

  • 浮动

    float 定义元素那个方向浮动。元素浮动以后会从文档流中删除。

    float: left; 元素向左浮动。

    float: right; 元素向右浮动。

  • 清除浮动

    1. 额外标签清除浮动，在最后一个浮动元素后面添加一个空元素设置样式 clear: both;
    2. 浮动元素的父级使用overflow: hidden;
    3. 用的最多的是伪元素清除浮动。 
      .clearfix:after {
  content: "";
  display: table;
  clear: both;
}

.clearfix {
  *zoom: 1;
}
