CSS 连接

  • 连接的四种状态

    a:link 没有被访问的连接
    a:visited 已经被访问过的连接
    a:hover 鼠标指针移动到连接上
    a:active 正在被点击的连接

    在为连接的不同状态设置样式的时候，需要按照以上表格中的顺序。

  • 文本修饰

    text-decoration 多用于去掉连接的下划线。

    a:link {text-decoration: none;}
a:visited {text-decoration: none;}
a:hover {text-decoration: underline;}
a:active {text-decoration: underline;}

  • 背景色

    background-color 设置连接的背景色。
