CSS 表格

  • 表格的属性

    border-collapse 合并表格的边框。
    border-spacing 设置单元格的距离。
    caption-side 设置表格标题的位置。
    empty-cells 设置是否显示空单元格。
    table-layout 设置单元、行还有列的算法

  • 折叠边框

    border-collapse 设置边框是否合并。

    table {border-collapse: collapse;}

  • 表格文本对齐

    text-align 设置水平对齐方式，比如左对齐、右对齐或者居中对齐。

    vertical-align 设置垂直对齐方式，例如top、bottom或者middle。

  • 显示空的单元格

    empty-cells 设置是否显示空的单元格。

    empty-cells: hide; 不在空白的单元格四边绘制边框。
    empty-cells: show; 在空白单元格周围绘制边框。默认值。
