CSS 文本

CSS 文本属性

color 设置文本颜色
direction 设置文本方向
line-height 设置行高
letter-spacing 设置字母间距
text-align 对齐元素中的文本
text-decoration 文本装饰
text-indent 首行缩进
text-shadow 设置文本阴影
text-transform 控制文本大小写
white-space 设置元素中空白的处理方式
word-spacing 设置单词间距

  • 缩进文本

    text-indent 可以实现文本缩进。

    text-indent 只可以给块级元素设置缩进。

    使用负值

    text-indent 可以使用负值，如果一个段落设置了负值，那么首行文字可能会超出容器的左边界。为了避免出现这种显示问题，可以设置一些内边距或者外边距。

    使用百分比

    text-indet 可以使用长度单位，也可以使用百分比。百分比是相对容器的宽度。

    继承

    text-indent 属性可以实现继承。

  • 水平对齐

    text-align 可以设置一个容器中文本相互间的对齐方式。

    值left、right、center 分别设置左、右、居中对齐。

    justify

    justify 设置文本的行的左右两端本别放在容器的内边界上。用户代理确定对齐文本如何拉伸，而不是 CSS。

    text-align: justify; 表现形式如下：

    There is clearly a need for CSS to be taken seriously by graphic artists. The Zen Garden aims to excite, inspire, and encourage participation. To begin, view some of the existing designs in the list. Clicking on any one will load the style sheet into this very page. The code remains the same, the only thing that has changed is the external .css file. Yes, really

    text-align: left; 可以和justify进行一下比较。

    There is clearly a need for CSS to be taken seriously by graphic artists. The Zen Garden aims to excite, inspire, and encourage participation. To begin, view some of the existing designs in the list. Clicking on any one will load the style sheet into this very page. The code remains the same, the only thing that has changed is the external .css file. Yes, really

  • 字间隔

    word-spacing 可以改变单词之间的标准间隔。默认值是 normal 和设置值为 0 效果是一样的

    word-spacing 接受负值，正值间隔增加，负值将拉近间隔。

    正值表现如下（word-spacing: 30px;）：

    This is a paragraph. The spaces between words will be increased.

    负值是这样的（word-spacing: -0.5em;）：

    This is a paragraph. The spaces between words will be increased.

  • 字母间隔

    letter-spacing 修改的是字符或者字母的间隔。

    letter-spacing: -0.5em;

    This is header 1

    letter-spacing: 20px;

    This is header 1

  • 字符转换

    text-transform 处理文本的大小写。

    默认值 none 对文本不做任何改动，uppercase 和 lowercase将文本转换为全大写和全小写。最后capitalize 只对每个单词的首字母大写。

  • 文本修饰

    text-decoration 属性有5个值。

    分别是 none、underline、overline、line-through和blink。

    underline 会对元素加下划线，overline会给元素加上划线，line-through则对文本中间画一个贯穿线，blink会让文本闪烁。

    text-decoration 的值会替换而不会累加，因此下面的规则 class 为 stricken 的 h2 都只有一条贯穿线，而没有上划线和下划线。

    h2.stricken {text-decoration: line-through;}
h2 {text-decoration: underline overline;}

  • 处理空白字符

    white-space 属性设置用户代理对文档空格、换行和 tab 字符的处理。

    下表总结了 white-space 属性的表现形式：

    normal 合并空白符 忽略换行符 允许自动换行
    nowrap 合并空白符 忽略换行符 不允许自动换行
    pre 保留空白符 保留换行符 不允许自动换行
    pre-line 合并空白符 保留换行符 允许自动换行
    pre-wrap 保留空白符 保留换行符 允许自动换行

  • 文本方向

    direction 设置块级元素中文本的书写方向。

    可能的值

    ltr 默认。文本从左向右。
    rtl 文本从右向左。
    inherit 继承父级的 derection 属性的值。
